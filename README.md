# macOS Update Script

A simple bash script to check and install macOS updates.
A safe, colorful, and automated **macOS update script** with optional Homebrew updates.

---

## Features

-  Checks for macOS updates  
-  Installs all available updates automatically  
-  Detects if a restart is required (manual)  
-  Updates Homebrew packages if installed  
-  Logs all actions to `mac-update.log` on your Desktop  
-  Color-coded terminal output for better readability  

---

## Usage

chmod +x update.sh
./update.sh
